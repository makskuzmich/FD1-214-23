Тег picture

Этот тег предназначен для помощи адаптива картинов для мобильной версии.
Внутри содержится тег "img" и теги "source". Количество тегов "source" зависит от задач.
Писать медиа запросы можно прямо в атрибуте "Media" тега "source".
Огромный плюс этого тега заключается в экономии ресурсов,он будет подгружать картинку необходимую для девайса.
Причём подгружаться будет только та картинка,которая подходит под медиазапрос.
